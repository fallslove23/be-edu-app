# 드롭다운 옵션 관리 시스템 - 사용 가이드

## 🎉 완성된 기능

일반 관리자(비개발자)가 **웹 브라우저에서 직접** 드롭다운 옵션을 관리할 수 있는 시스템이 완성되었습니다!

---

## 📍 접속 방법

### 1. 로그인
- 관리자 또는 매니저 권한으로 로그인

### 2. 메뉴 이동
- 좌측 사이드바 → **시스템** 섹션
- **드롭다운 옵션 관리** 클릭

---

## ✨ 주요 기능

### 1️⃣ 옵션 조회
- 카테고리별로 드롭다운 옵션 확인
- 검색 기능으로 빠른 옵션 찾기
- 옵션의 상태(활성/비활성) 확인

### 2️⃣ 옵션 추가
1. **새 옵션 추가** 버튼 클릭
2. 필수 정보 입력:
   - **값(Value)**: 시스템 내부값 (영문, 예: `mock_exam`)
   - **라벨(Label)**: 화면 표시 이름 (한글 가능, 예: `모의고사`)
3. 선택 정보 입력:
   - **설명**: 옵션에 대한 상세 설명
   - **아이콘**: 이모지 선택 (30개 이상 제공)
   - **색상**: 10가지 색상 중 선택
4. **추가** 버튼 클릭

### 3️⃣ 옵션 수정
1. 수정하려는 옵션의 **연필(✏️) 아이콘** 클릭
2. 정보 수정:
   - ⚠️ **값(Value)**은 수정 불가 (데이터 무결성 보호)
   - ✅ 라벨, 설명, 아이콘, 색상은 자유롭게 수정 가능
3. **수정** 버튼 클릭

### 4️⃣ 옵션 활성화/비활성화
- **눈(👁️) 아이콘** 클릭
- 비활성화된 옵션은:
  - 기존 데이터에는 영향 없음
  - 새로운 입력 시에만 선택 불가
  - 언제든 다시 활성화 가능

### 5️⃣ 옵션 순서 변경
- **위/아래 화살표(⬆️⬇️)** 버튼으로 순서 조정
- 드롭다운 메뉴에 표시되는 순서가 변경됨

### 6️⃣ 옵션 삭제
- **휴지통(🗑️) 아이콘** 클릭
- ⚠️ **시스템 옵션**은 삭제 불가 (수정만 가능)
- 일반 옵션만 삭제 가능

## 🎨 관리 가능한 드롭다운 카테고리

| 카테고리 | 설명 | 예시 |
|---------|------|------|
| **시험 유형** | 시험의 종류 구분 | 최종평가, 중간평가, 퀴즈 등 |
| **시험 상태** | 시험 진행 상태 | 준비중, 발행됨, 진행중 등 |
| **문제 유형** | 문제 출제 방식 | 객관식, O/X, 단답형 등 |
| **난이도** | 문제 난이도 | 쉬움, 보통, 어려움 |
| **과정 상태** | 과정 진행 상태 | 계획중, 모집중, 진행중 등 |
| **출석 상태** | 출석 구분 | 출석, 지각, 결석, 공결 |
| **사용자 역할** | 권한 구분 | 관리자, 매니저, 강사 등 |
| **평가 유형** | 평가 방식 | 이론, 실습, 프로젝트 등 |
| **성적 등급** | 성적 등급 | A+, A, B+, B, C, F |

## 💡 실전 사용 예시

### 예시 1: 새로운 시험 유형 추가

**시나리오**: "깜짝 퀴즈" 시험 유형을 추가하고 싶어요!

1. **시험 유형** 카테고리 선택
2. **새 옵션 추가** 클릭
3. 정보 입력:
   - 값: `surprise_quiz`
   - 라벨: `깜짝 퀴즈`
   - 설명: `사전 공지 없이 진행되는 퀴즈`
   - 아이콘: `⚡` 선택
   - 색상: `orange` 선택
4. **추가** 클릭

→ 즉시 시험 생성 화면에서 "깜짝 퀴즈" 선택 가능!

### 예시 2: 기존 옵션 수정

**시나리오**: "최종평가"를 "기말고사"로 변경하고 싶어요!

1. **시험 유형** 카테고리 선택
2. "최종평가" 옵션 찾기
3. **연필 아이콘(✏️)** 클릭
4. 라벨 변경: `최종평가` → `기말고사`
5. 아이콘 변경: `🎯` → `📝` (선택사항)
6. **수정** 클릭

→ 시스템 전체에서 "최종평가"가 "기말고사"로 표시됨!

### 예시 3: 임시로 옵션 숨기기

**시나리오**: "일일평가"를 일시적으로 사용 안 하고 싶어요!

1. **시험 유형** 카테고리 선택
2. "일일평가" 옵션 찾기
3. **눈 아이콘(👁️)** 클릭 → 비활성화
4. 나중에 다시 사용하려면 다시 클릭

→ 새로운 시험 생성 시 "일일평가" 선택 불가<br>
→ 기존 데이터는 그대로 유지됨

## ⚠️ 주의사항

### 🔒 시스템 옵션
- **"시스템"** 배지가 있는 옵션:
  - ✅ 수정 가능 (라벨, 아이콘, 색상 변경 가능)
  - ❌ 삭제 불가 (시스템 안정성 보호)

### 📊 값(Value) 변경 금지
- 값(Value)은 **데이터베이스 키**로 사용됨
- 한번 생성 후에는 **절대 변경 불가**
- 이유: 기존 데이터와의 연결이 끊어질 수 있음

### 💾 데이터 안전성
- 옵션 삭제 시 확인 메시지 표시
- 모든 변경사항은 **자동으로 이력 저장**
- 변경 이력 조회 기능 (향후 추가 예정)

## 🆘 문제 해결

### Q: 옵션을 추가했는데 드롭다운에 안 보여요
**A**: 페이지를 새로고침(F5)하거나 해당 화면을 다시 열어주세요.

### Q: 시스템 옵션을 삭제하고 싶어요
**A**: 시스템 옵션은 삭제할 수 없습니다. 대신 **비활성화**를 사용하세요.

### Q: 실수로 잘못 수정했어요!
**A**: 다시 수정하거나, 변경 이력 기능을 사용하여 복구하세요 (향후 추가 예정).

### Q: 옵션 순서가 이상해요
**A**: 화살표 버튼으로 원하는 순서로 재배치하세요.
